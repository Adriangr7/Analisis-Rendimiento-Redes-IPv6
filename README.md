# Analisis-de-rendimiento-de-redes-IPv6-TFG
Proyecto de fin de grado enfocado en la comparación del rendimiento de IPv6 nativo proporcionado por el  operador frente a soluciones alternativas como túneles y VPNs en redes sin soporte IPv6. El estudio analiza  parámetros de red como latencia, pérdida de paquetes y tiempos de resolución DNS. Calificación obtenida: 10  (sobresaliente).
